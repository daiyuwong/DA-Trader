# DA-Trader
First version of meme coin trading bot that operates based on pumpfun data. 


How to run in english

1. Open CMD in DA Trader directory.
2. pip install -r requirements.txt
3. python main.py

Comment run en FR si tu es con

1. Ouvre CMD dans le fichier DA Trader
2. pip install -r requirements.txt
3. python main.py
